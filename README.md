# Lecture-Voice-to-Notes-Generator-IBM-AI-ML-internship-
AI-powered app that transcribes lecture audio to text using speech-to-text (Whisper) and generates concise study notes, quizzes, and flashcards via generative AI (Hugging Face models). Built with Python, Streamlit for easy deployment—perfect for students multitasking in class.
Key FeaturesAudio Transcription: Converts 30-60 minute lecture audio files (WAV/MP3) to accurate text using Whisper (90%+ accuracy on clear audio)Intelligent Summarization: Generates structured bullet-point study notes using extractive/abstractive summarization techniquesAuto-Generated Quizzes: Creates 5-10 targeted multiple-choice questions from lecture content for active recallFlashcard Generation: Exports key concepts as Anki-compatible flashcards for spaced repetitionMulti-Format Export: Download notes as PDF, Markdown, or copy to clipboardReal-time Processing: Complete pipeline (transcription → notes → quizzes) in under 2 minutesStudent-First UI: Simple drag-and-drop interface leveraging your frontend development skills (HTML/CSS/JavaScript + Streamlit)
Future Features:Real-time live transcription (WebRTC + Whisper) Multi-speaker diarization Multi-language support (Whisper's 99 languages),Mobile PWA deployment,Study analytics dashboard
